Weather Dashboard

 Project Overview:

 Weather Dashboard is a responsive web application built using React.js.
 It allows users to search and view weather information based on city.
 
 The application fetches weather data from an external API and displays:
 
Temperature

Humidity

Pressure

Weather Condition
